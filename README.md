# FastAPI CRUD Learning Project

This is a backend project built using **FastAPI** as a hands-on learning exercise to understand how modern APIs are structured and implemented.

## 🚀 What I Learned

- Implemented the fundamentals of FastAPI
- Set up and configured a database connection
- Designed database models using ORM classes
- Integrated models with application logic
- Built and tested complete CRUD (Create, Read, Update, Delete) operations

## 🎯 Project Purpose

This project was created to gain practical experience in building and structuring backend APIs that can be integrated into real-world applications.

## 🖥️ Frontend Resource

The frontend used for testing and integration was provided as a learning resource by Telusko (YouTube).  
The tutorials explained the core concepts clearly and helped in understanding how to connect a frontend with a FastAPI backend.
